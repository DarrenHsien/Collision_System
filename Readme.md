# 碰撞分析 (Collision Analysis)

本專案使用 AABB vs AABB + BVH + GJK + EPA +  Voxelization + SDF 組合 來解決物體碰撞問題


## 功能特色

- 使用 Python 實現遺傳演算法。
- 可自訂城市數量與座標。
- 視覺化呈現初始路徑與優化後的最佳路徑。
- 可調整遺傳演算法的各項參數 (如：族群大小、突變率、交叉率)。

## 環境建置與安裝


```bash
git clone https://github.com/DarrenHsien/Collision_System.git
cd collision_system
```


## 如何使用

直接執行主程式 `main.py` 即可開始進行路線優化。

```bash
python main.py
```
